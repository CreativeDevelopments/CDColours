# Installation
```
npm i cdcolours
```

<br>

# Setup

## JavaScript
```js
// File Name - example.js

const colour = require('cdcolours');

console.log(colour('Hello World', { textColour: 'green', backgroundColour: 'cyan'}));
```

## TypeScript
```ts
// File Name - example.ts

import colour from 'cdcolours';

console.log(colour('Hello World', { textColour: 'green', backgroundColour: 'cyan'}));
```
